Tentang Sistem

🧬 Metode: Backpropagation ANN
Sistem menggunakan Artificial Neural Network dengan algoritma Backpropagation untuk mempelajari pola hubungan antara variabel input (BB, TB, umur, jenis kelamin) terhadap status gizi dan kebutuhan kalori.
📐 Parameter Input
Berat Badan (kg) · Tinggi Badan (cm) · Umur (tahun) · Jenis Kelamin (L/P)
🏷️ Output Sistem
Status Gizi: Kurus · Normal · Overweight · Obesitas
Rekomendasi Kalori Harian (kkal/hari) disesuaikan kondisi tubuh.
